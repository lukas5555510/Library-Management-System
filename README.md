Console-based Library Management System in Java. Goal of this project is to learn and try in practice some java features and SOLID principles.


# Features
* Adding book
* Adding member
* Borrowing book
* Returning book
* Displaying books
* Displaying members
* Saving data to file
* Restoring data from file
